## 4-in-a-row

### Цель игры

Собрать 4 фишки в ряд. Считаются комбинации: горизонталь, вертикаль, диагональ.

### Как это сделать

Кликайте в любой столбец, фишка будет падать сверху вниз до доступной ячейки. Когда столбец заполнится, в него больше не кинуть фишку.

### Начало игры

Игра доступна в двух вариантах:

- Игрок + игрок
- Игрок + ПК

P.S.: Пока что сменить режим игры можно только программно в компоненте App (isVsComputer), в будущем найду место, куда поставить кнопку для смены режимов игры.

#### Кто ходит первым

Первым ходит игрок. (Если игра игрок + игрок, первым ходит всегда игрок 1). (Если игра игрок + ПК, первым ходит всегда игрок).

### Когда подошел ваш ход

Отображение хода показывается в шапке. Если выбранная вами фишка подсвечивается снизу, значит сейчас ваш ход.

### Основной функционал

- Генерация поля размером 7 на 6 (уже есть задел на изменение размеров поля)
- Выбор игровой фишки из 4 цветовых вариантов
- Выбор режима игры (Игрок + игрок, Игрок + ПК)
- Автоматическая остановка игры, если поле заполнено (ничья) либо найдена победная комбинация
- Анимация победной комбинации
- Досрочное завершение игры
